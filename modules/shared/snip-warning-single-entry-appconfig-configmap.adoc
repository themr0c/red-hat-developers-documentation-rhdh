:_mod-docs-content-type: SNIPPET
[WARNING]
====
Each `app-config` config map must contain only one data entry.
If a config map contains multiple `{my-app-config-file}` files, their keys are sorted alphabetically, which might not match your intended merge order.
To control the merge order, split multi-entry config maps into separate single-entry config maps and reference each one in the `spec.application.appConfig.configMaps` field of your `{product-custom-resource-type}` custom resource in the intended order.
====
